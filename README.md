# 전기기사 실제 기출 모바일 문제은행

공개 GitHub Pages에는 앱 껍데기만 포함하고, 사용자 소유 복원 기출은 `private-banks/*.bank`로 분리한 개인용 PWA입니다.

## 사용

1. GitHub Pages 주소를 휴대폰에서 엽니다.
2. **기출 파일 불러오기**를 눌러 `private-banks` 폴더의 원하는 `.bank` 파일을 고릅니다.
3. 가져온 회차는 해당 브라우저의 IndexedDB에만 저장됩니다.
4. 회차별, 100문항 실전, 과목별 랜덤, 오답 다시 풀기를 사용합니다.
5. 결과에서 Markdown을 복사하거나 `.md`로 내려받아 Codex/Notion에 붙여 넣습니다.

## 시험 설정

- 필기 5과목, 과목당 20문항·30분
- 실전 모드 100문항·150분
- 과목당 40점 이상 및 전과목 평균 60점 이상 기준
- 목표일 2026-09-01

## 비공개 데이터

`private-banks/`, `*.bank`, PDF, OCR 중간 파일은 `.gitignore`로 공개 저장소에서 제외됩니다. 브라우저 데이터를 지우면 가져온 회차도 삭제되므로 `.bank` 원본을 개인 저장소에 백업하세요.

## 정답 검수 정책

문항·해설 이미지는 실제 PDF에서 분리했습니다. OCR로 정답 표식이 명확한 문항만 자동 채점하며, 애매한 표식은 정답을 추측하지 않고 **검수 대기**로 표시하여 점수에서 제외합니다. 검수 대기 문항도 선택 직후 원문 정답·해설을 확인할 수 있습니다.
